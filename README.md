Android Gallery App

A mobile application developed in Android Studio that displays images stored on a device and allows users to browse and view photos in full-screen mode.

Features
Displays images stored on the device
Grid-based gallery interface
Full-screen image viewer
Image metadata handling
Responsive user interface
Modern Android permission support
Technologies Used
Kotlin
Android Studio
Android SDK
RecyclerView
MediaStore API
Project Overview

This project was developed as part of my Bachelor of Computer Science studies. The application scans images stored on a device, presents them in a gallery layout, and allows users to select and view images individually.

The project focused on Android application development, image handling, user interface design, and working with device media storage.
